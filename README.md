# node - Notas

Aplicacion para generar tablas de multiplicar.

```
Options:
      --help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -b, --base     Numero para el calculo                      [number] [required]
  -l, --listar   Indica si imprime tabla en consola   [boolean] [default: false]
  -t, --tope     Numero maximo para el calculo               [number] [required]

```
